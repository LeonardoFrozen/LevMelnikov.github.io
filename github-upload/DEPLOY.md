# Публикация сайта

Сайт собран как статический проект: `index.html`, `roots-project.html`, `styles.css` и папка `assets`.

## Быстрый вариант: Netlify Drop

1. Открой Netlify Drop: https://app.netlify.com/drop
2. Перетащи туда всю папку проекта или архив `portfolio-site.zip`.
3. Netlify выдаст публичную ссылку, которую можно вставить в CV.

## Постоянный вариант: Netlify или GitHub Pages

Для постоянной ссылки лучше загрузить проект в GitHub-репозиторий и подключить его к Netlify или GitHub Pages. Тогда после правок сайт можно будет обновлять через репозиторий.

## Важно перед финальной публикацией

В проекте еще есть изображения, которые подключены по ссылкам Figma MCP. Для надежной публичной версии лучше сохранить эти изображения локально в `assets` и заменить внешние ссылки на локальные пути. Иначе часть визуала может пропасть, если Figma-ссылки перестанут быть доступными.
